Gradientes

Css define três topos de gradientes

• Gradientes lineares (desce/cima/esquerda/direita/diagonalmente)
• Gradientes radiais (definidos pelo seu centro)
• Gradientes Cônicos (girados em torno de um ponto central)

Sintaxe

background-image: linear-gradient(direção, cor 1, cor 2, ...)

De cima para baixo

background-image: linear-gradient(red,yellow);

Da esquerda para a direita

background-image: linear-gradient(to right,red,yellow);

superior esquerdo para inferior direito.

background-image: linear-gradient(to right bottom ,red,yellow);
